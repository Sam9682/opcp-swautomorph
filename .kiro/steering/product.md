# Product Overview

OPCP-SwAutoMorph is a centralized application deployment and management platform designed for GenAI agents. It enables autonomous deployment, lifecycle management, and SSO authentication for web applications.

## Core Purpose

Enable GenAI agents to autonomously deploy, manage, and access web applications without human intervention.

## Key Features

- Multi-interface access: Web dashboard, CLI, REST API, MCP protocol
- User authentication with Gitea integration and SSO token-based auth
- Multi-server deployment with automatic capacity management
- Dynamic Nginx reverse proxy configuration per user/app
- Virtual AI agents: Developer and Operations assistants with context-aware prompts
- PostgreSQL database with connection pooling for enterprise-grade performance
- Automated hourly backups with S3 sync capability
- Billing and cost tracking with activity logging
- ModSecurity WAF protection with OWASP CRS rules
- Multi-language support (English/French)
- Real-time streaming APIs with Server-Sent Events

## Target Users

- GenAI agents requiring autonomous application deployment
- Development teams needing centralized app management
- Operations teams managing multi-server deployments
